# 🏦 Bank Account Simulator (C/C++)

## 📌 Project Overview
This project is a simple **Bank Account Simulator** built using **C/C++**.  
The goal is to practice and demonstrate core **Object-Oriented Programming (OOP)** concepts such as:

- Constructors
- Encapsulation
- State Management

The program allows users to perform multiple banking transactions like deposits, withdrawals, and balance checks.

---

# 🚀 Features

✅ Create a bank account with:
- Account Number
- Account Holder Name
- Balance

✅ Perform banking operations:
- Deposit Money
- Withdraw Money
- Display Balance

✅ Additional Validations:
- Prevent withdrawal if balance is insufficient
- Maintain minimum balance rule

✅ Supports multiple transactions in one run

---

# 🛠️ Concepts Used

## 1. Constructors
Used to initialize account details when an object is created.

## 2. Encapsulation
Bank account data is kept private and accessed through public methods.

## 3. State Management
The balance changes dynamically after every transaction.
